My Projects

Navigation System

I built the backend of a simple navigation system using Dijkstra's Algorithm to find the shortest path between locations. I represented the map as a weighted graph and used Java's PriorityQueue for efficient pathfinding. The program reads map data from a file and returns the shortest route and distance between any two points. This project helped me strengthen my understanding of graph algorithms, Java collections, and modular object-oriented design.
